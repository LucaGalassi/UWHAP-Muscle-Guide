# Models Folder

Place a rigged GLTF/GLB skeleton here as `skeleton.glb`.

- Licensing: Prefer CC0 (no attribution required). If CC-BY, add credits in the main README.
- Recommended path: `public/models/skeleton.glb`
- The app will attempt to load it at runtime via `/models/skeleton.glb`.
- If missing, the viewer falls back to a simple box-based rig.

Notes:
- Typical bones mapped (examples): `Shoulder_R`, `Elbow_R`, `Forearm_R`, `Hip_R`.
